Tabbo WOFF2 Font Preview

Each cell shows a glyph rendered at 36px with its PK code (decimal) and Unicode codepoint below. ASCII zone glyphs (fret letters, punctuation) use their natural Unicode codepoints. Numeric PK codes are mapped to PUA: U+E000 + pk_code. Winding direction is normalised via FontForge correctDirection() on all glyphs. If a cell is blank, that glyph is not defined in this font variant.
